Python基础

列表(list)：有序集合，可以随时添加和删除其中元素。 

classmates = ['1','2','3']

元组(tuple): 不可修改的列表，经常用于仅遍历的情况。 

classmates = ('1','2','3')

字典(dict): 存储“键-值”对数据，键不可重复。 

keys = ['a','b','c','d']  
values = [1,3,7,10]  
dict1 = dict(zip(keys,values))

students = {'wyb':1 'zjn':2}

集合(set): 集合元素不允许重复 

set1 = {1,3,4,5}

set可看成数学意义上的无序和无重复元素的集合，故可进行数学上的交集，并集运算。 

s1 = set([1,2,3])  
s2 = set([2,3,4])  
s1 & s2
s1 | s2

函数

返回值是tuple。 

def my_function():  
    pass

定义默认参数要牢记一点：默认参数必须指向不变对象！

可变参数 允许你传入0个或任意个参数，这些可变参数在函数调用时自动组装为一个tuple 

def my_fun(*number):  
    pass

关键字参数 允许你传入0个或任意个含参数名的参数，这些关键字参数在函数内部自动组装为一个dict 

def my_fun(name, age, **kw):
    pass

命名关键字参数 

def my_fun(name, age，*， city， home):
    pass

参数组合 参数定义的顺序必须是：必选参数、默认参数、可变参数、命名关键字参数和关键字参数。
