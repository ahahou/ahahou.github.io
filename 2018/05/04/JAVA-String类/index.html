JAVA_String类

发表于 2018-05-04
字数统计: 408 字 | 阅读时长 ≈ 2 分钟

在程序设计中经常会设计处理和字符串有关的算法，而java中提供了专门用来处理字符序列的String类。

一下就介绍一下java中的String类。

String类，位于java.lang包中。
String类被定义为final类 ，故不可以有子类。

1.常量对象
用双引号括起来的字符序列。
String常量也是对象
如”你好”，”12”
Java把用户程序中的String常量放入常量池，常量池中的数据在程序运行期间不能改变。

2.String对象
String类声明创建对象。
如：

String s = new String("cool");  
String t = new String("cool");

**注意**  
s == t 结果是false。  
s 和 t 是不同的引用，尽管实体相同但是会出现上面的情况。  
同时用户是无法输出String对象的引用的。system.out.println(s); 输出的是cool。

3.引用String常量 

String s1,s2;  
s1 = "hi";  
s2 = "hi";  

s1 == s2 返回时true。 实际上时对常量对象的引用，可以这样理解，"hi"本来自己有自己的专属名字，但别人不知道，他也不愿告诉别人，于是就给他起个别名，相当与它的名字，故s1 和 s2都代表它的名字，都指向它之个人。故s1 == s2 是true。

String类的构造 

String s = new String("");
String t = new String(String s);
String s = new string(char a[]);
String s = new string(char a[], int b, ...);
