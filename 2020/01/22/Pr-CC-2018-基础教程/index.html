Pr CC 2018 基础教程

发表于 2020-01-22 | 分类于 教程 ， 视频剪辑
字数统计: 680 字 | 阅读时长 ≈ 2 分钟

本文是为了快速上手Pr而写的一篇基础性指导文章，适合一个新手小白，快速的了解Pr剪辑的一些步骤和简单的进行一些操作，关于深层次的说明，更深入的视频剪辑技巧，本文不做过多的叙述。我是用的是Pr Pro CC 2018，其他版本一些操作大同小异，可以作为参考。

制作一个视频

新建项目

渲染程序：如果是NVIDIA显卡选择CUDA加速。
其它均选默认。

新建工程

有许多窗口。 万一不小心关掉了面板，可以选择窗口工作区，重置为保存布局。

新建序列

序列相当于一个容器，对要剪辑的素材进行存放。
文件->新建序列 ctrl +n
设置
时基：一般25帧每秒够用
像素长宽比：方形像素
采样率：44100Hz

导入文件 ctlr + i 也可以直接拖入

导出文件 ctlr + m
格式： 若导出MP4格式 选择h.264
匹配源：高比特率 视频清晰度和比特率有关。
目标比特率和最大比特率默认就好了。

剪辑技巧及剪辑工具

~键 放大某个窗口。
文件 项目设置 常规中可以修改项目。
视频预览窗口 出入点设置。使用i 设置入点，使用o设置出点
视频音频都要，就按住画面推入到时间轴中
剃刀工具 C 选择工具 V
一个视频片段变长，使用波纹编辑工具B，可以不用麻烦移动后面的视频片段。
滚动 编辑工具 N 会影响其他的视频片段
比率拉伸工具 R 慢放的好工具

速度和关键帧

慢放：右键片段 速度持续时间 若100帧改为25帧，即可以慢放4倍
高帧率后期慢放才会流畅
右键视频文件，修改 解释素材 也可实现上述效果

快放：右键 速度持续时间 把帧升高
在预览中快放 空格播放 按l是快放 按j是倒放 不影响视频文件

效果控件shift +5 可以弹出此面板 时间重映射 比较好用
选择要改变速度开始的一帧，添加关键帧，在速度结束改变的一帧，添加关键帧
可以在时间轴中把视频拉大，即竖直拉
然后右键 显示剪辑关键帧，时间重映射，速度
可以使用钢笔工具P 可以快速打关键帧
将关键帧中间往下压， 慢放
